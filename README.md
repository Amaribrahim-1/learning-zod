# Next.js & TypeScript Practice Project

This is a small, lightweight practice project built specifically to demonstrate proficiency in modern frontend technologies. It is not a production application, but rather a practical playground to showcase specific skills.

## 🎯 Core Skills Demonstrated

- **Next.js (App Router)**: Utilizing Server Components, Server Actions, and modern data-fetching patterns.
- **TypeScript**: Ensuring strict type safety, utilizing interfaces, and employing utility types for robust code.
- **React Hook Form (RHF)**: Managing complex form state, validation, and submissions efficiently.
- **Zod**: Implementing schema-based validation integrated seamlessly with RHF for robust type-safe forms.

_Note: This repository serves primarily as a portfolio piece to prove my ability to integrate and effectively use these specific tools together._
